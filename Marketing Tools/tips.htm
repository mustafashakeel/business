Tips for completing forms

The forms

Forms in this kit are supplied in two formats: Acrobat (.PDF) and Microsoft Word (.DOC).

Microsoft Word:

Blank forms suitable for completing in your word processor are supplied in this format. You can access these forms either through a folder installed on your Windows desktop when you installed this software, or through the Windows Start menu.

Each of the Word forms is quite easy to modify, so you could add your company logo to each, or edit some files to add or remove items.

Acrobat .PDF:

The Portable Document Format (PDF) used by Acrobat software from Adobe, offers a highly reliable way of seeing forms the way they should look.

You can print the .PDF forms but you cannot complete (fill in) the blank forms on-screen. Use this format if you want to hand-write your forms or if they will be completed using a typewriter.

Even if you intend using the Word forms, we advise printing at least one set of blank forms using Acrobat, so that you have a visual reference to the layout of each form. This can be important: word processing software can cause lines to wrap wrongly, or margins to move.

If you do not have Adobe Acrobat Reader installed on your computer:

  • if you bought this kit on a CD-ROM, please insert the CD-ROM in your computer and when the screen appears, select the option to install a free copy of Adobe Reader.
  • if you downloaded this kit, please return to the previous page and use the link in the right side column.

Using WordPerfect, StarOffice and other word processors


We do not supply forms prepared with WordPerfect. However, any recent version of WordPerfect will import (open) and display for editing a Microsoft Word format file. If you choose this approach, please make sure to print sample copies of the forms using the Acrobat format, so that you have a visual reference to the required layout.

StarOffice and OpenOffice offer compatibility with Microsoft Office applications and should open .DOC files without a problem. If you choose this approach, please make sure to print sample copies of the forms using the Acrobat format, so that you have a visual reference to the required layout.

Other word processing software may be able to import or open Microsoft Word (.DOC) format files. Be sure to use forms examples printed with Acrobat for comparison, to ensure the word processor you are using is not changing the formatting of the forms.
